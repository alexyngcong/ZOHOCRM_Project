Design Principles

Built on seven unbreakable rules

Every decision in the MICS system follows these core principles. They are enforced by Zoho Flow automation and cannot be bypassed.

◉
Employee Code Attribution
Every record and action carries a traceable Employee Code. Shared logins preserved; individual accountability guaranteed.
◐
Mandatory Payment Gate
No Work Order is created and no execution begins until the agreed deposit is reconciled in Zoho Books.
◆
Ownership Set at Creation
Manually created records belong to their creator. Web inquiries are round-robined or assigned to first responder.
∞
Permanent Source Attribution
Lead Source Owner and External Referral never change. Marketing ROI and referral tracking remain accurate.
▶
Execution Sees Everything
Assigned delivery team has FULL visibility into signed contract, fees, portal credentials and client history.
⎋
Every Action Audited
Immutable log of all views, edits, exports and credential reveals. 7-year retention, keyed by Employee Code.
